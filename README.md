# AMRDB - Antimicrobial Resistance Database

**A comprehensive data portal for antimicrobial resistance (AMR) insights in Uganda**

Visit: [https://amrdb.idi.co.ug](https://amrdb.idi.co.ug)

---


## What is AMRDB?

AMRDB is a dedicated data portal that facilitates access to antimicrobial resistance (AMR) data and insights to enhance public health initiatives in Uganda. The platform connects users to a central AMR data lakehouse through an intuitive interface, enabling researchers, healthcare professionals, and policymakers to access, visualize, and analyze critical AMR data.

## Why AMRDB?

Antimicrobial resistance is a growing public health threat. AMRDB addresses this challenge by:

- **Centralizing AMR Data**: Bringing together diverse AMR-related datasets in one accessible location
- **Facilitating Research**: Providing researchers with tools to analyze trends and patterns
- **Informing Policy**: Offering visualizations that help policymakers make evidence-based decisions
- **Promoting Collaboration**: Enabling data sharing within the research community through controlled access
- **Building Capacity**: Supporting LMIC contexts with user-friendly analysis tools

## Platform Benefits

AMRDB bridges the gap between raw AMR data and actionable insights, serving as Uganda's central hub for antimicrobial resistance information. By connecting the data lakehouse to an intuitive interface, the platform delivers measurable value across multiple stakeholder groups.

### For Individual Users
- **Easy Access**: User-friendly interface for complex data
- **Time Saving**: Pre-processed and quality-checked datasets
- **Comprehensive**: Multiple data sources in one location
- **Secure**: Protected data with proper governance
- **Insightful**: Built-in visualization and analysis tools

### For the Research Community
- **Collaboration**: Facilitated data sharing among researchers
- **Standardization**: Common data formats and quality standards
- **Transparency**: Clear data provenance and methodology
- **Innovation**: Platform for showcasing student and researcher ML models
- **Impact**: Enhanced research output and publications

### For Public Health
- **Evidence-Based**: Data-driven decision making
- **Timely**: Current AMR trends and patterns
- **Actionable**: Insights ready for intervention planning
- **Comprehensive**: Multi-dimensional view of AMR landscape
- **Sustainable**: Long-term data resource for Uganda
